Features

DMC Features

  • Document Capturing

    Capture scanned documents to send to DMC
    Ability to change scanned pages order
    Ability to merge several scanned pages into one document
    Capture barcode to retrieve document

  • Integration

    Ability to send documents to DMC via different integration methods:
    Windows context menu, by right clicking directly from windows on a
    document to send directly to DMC :
    Office integration, to upload any document directly to DMC
    Outlook integration
    Office web apps integration

  • Optical character recognition (OCR)

    The process of converting images containing typed or clearly written text into machine-encoded text that is editable and searchable.
    Application interface offers Arabic and English versions.

  • Indexing

    Indexing of content of documents is offered in the DB, this grantees:
    Fast performance in the application.
    Fast Free text search on content of documents.

  • Security

    System is compatible with SharePoint security.
    All users are defined from active directory and added into SharePoint.
    Groups are pre-defined on SharePoint.
    Only users or groups who have access to a document can preview it or retrieve it in their search results.
    Others wonâ€™t be able to access documents.

  • Search Options

    The DMC search engine allows users to easily search documents based on the content inside. Here are some options of the search process:
    Quick Search Via input field, available in all site.
    Simple Search Via document ID or name or Barcode.
    Advanced Search Via different search criteria ( document ID, name, barcode, free content text, type, size, create date, update date, user or group, version, tags)
    Saved Searches Customizable for every user.
    Search using the search textbox which available in all the site.
    Advanced Search page
    Save Search options in saved Search list, to easily navigate to it when necessary.

  • Document Management

    In "Live" status, user can: View, Update, Edit, Link, Share, Lock, Move, Archive, Recycle, Print and Download.
    Also, user can Add comments, change privileges, trace version, add to Bookmarks and add to favorites.
    In "Recycled" status, user can: View, Delete Permanently, and Restore.
    In "Pending" status, user can view, update, delete document.

  • Document Version

    DMC helps you track changes made to a document. Automatically create new versions each time a file is modified to assure no critical data is lost through the editing process.
    Version assures work is never lost. And rolling back is as easy as selecting that document and choosing a version from a list of versions.

  • Records Management

    After duration predefined in settings set by administrator, documents are automatically transferred from:
    "Live" status to "Archived" status with their activity log.
    "Archived" to "Recycled" status with their activity log.
    Then finally deleted from "Recycled" status at the end.

  • Administration

    Admin has the privileges
    Change the values of durations of when to Archive, Recycle and purge documents.
    Set the pre-defined path of pending documents, either unique for all users or different per user
    Browse through uploaded documents to either approve or reject them to be deleted.
    Create tags, which can be linked to documents.

  • PDF conversion

    DMC has an add-on feature that converts documents when needed into secure PDF files. These PDF files has an advantage that user could not copy and paste from it, because simply its content are dealt with as images and not text.

  • Full Arabic Support

    DMC provides Full Arabic Support in its User Interface and Arabic Data Support in its Backend in the Database.

    DMC enables users in search to retrieve data easily without having to worry about Arabic complexity.
